# na3js-homework

[![build](https://travis-ci.org/ikatyang/na3js-homework.svg)](https://travis-ci.org/ikatyang/na3js-homework)

numerical analysis homework using three.js

## Development

require `RubyGems@stable`, `Bundler@stable`, `Node.js@>=6`, `Python@3.x`

#### Setup

```sh
npm install
```

#### Generate HTML

```sh
npm run build
```

#### Preview HTML

```sh
npm run preview
```
